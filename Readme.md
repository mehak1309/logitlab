# LLaMA

A minimal PyTorch implementation of a LLaMA-style language model, built from scratch.

---

- [x] RMSNorm
- [x] RotaryEmbedding (RoPE)
- [x] FeedForward (SwiGLU)
- [x] MultiHeadAttention (GQA)
- [x] Block
- [x] Llama
- [x] Grouped Query Attention
- [x] KV Cache
- [x] RoPE with start_pos
- [x] FlashAttention
- [x] Causal masking
- [x] Weight tying
- [x] Prefill + decode generate loop
